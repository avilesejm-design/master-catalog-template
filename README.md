# Master Catalog Template

Plantilla reutilizable de catálogo dinámico para pequeños emprendimientos.

## ¿Qué es?

Sistema "backend-less" que convierte Google Sheets en catálogo de productos con búsqueda en tiempo real. Ideal para negocios que quieren digitalizar su stock sin pagar mantenimiento.

## Características

- **Sin servidor**: Google Sheets como base de datos
- **Búsqueda instantánea**: Filtra productos sin recargar página
- **Responsive**: Funciona en celular y desktop
- **Deploy rápido**: GitHub Pages + Netlify en minutos
- **Gratis**: $0 costos de infraestructura

## ¿Quién lo usa?

- [Camisetas Icónicas](https://camisetasiconicas.netlify.app/) - 54+ productos

## Tecnologías

- Google Apps Script (API)
- Google Sheets (CMS)
- Vanilla JavaScript
- HTML/CSS

## Cómo usar

1. Hacé fork de este repo
2. Configurá tu Google Sheet con los campos: Nombre, Precio, Imagen, Categoría
3. Conectá el Apps Script
4. Deploy en GitHub Pages o Netlify
5. Listo
